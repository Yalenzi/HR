# 🏥 نظام إدارة الموارد البشرية المتقدم
## Advanced HR Management System

نظام شامل لإدارة الموارد البشرية مع مولد بطاقات متقدم ولوحة تحكم احترافية لمركز الخدمات الطبية الشرعية بمنطقة الحدود الشمالية.

---

## 🌟 **الميزات الرئيسية**

### 🎨 **مولد البطاقات المتقدم** (`extension-card-generator/`)
- **لوحة تحكم شاملة** مع إحصائيات وتحليلات
- **محرر مرئي** بتقنية السحب والإفلات
- **إدارة الحقول المخصصة** مع جميع العمليات CRUD
- **مكتبة الخلفيات** مع إمكانية الرفع والإدارة
- **نظام القوالب** مع الاستيراد والتصدير
- **محرر الإحداثيات المرئي** مع معاينة فورية

### 📊 **لوحة التحكم الإدارية**
- **إحصائيات شاملة** للبطاقات والاستخدام
- **إدارة البطاقات** مع البحث والفلترة
- **محرر الحقول المخصصة** مع التحقق من الصحة
- **إدارة القوالب** مع التخصيص الكامل
- **نظام النسخ الاحتياطي** والاستعادة

### 🎯 **النظام الأساسي** (Next.js)
- **قوالب التهنئة** للمناسبات المختلفة
- **قوالب التمديد** للقرارات الإدارية
- **شهادات الشهود** للموظفين
- **تصميم متجاوب** لجميع الأجهزة

---

## 🚀 **التشغيل السريع**

### **1. النظام الأساسي (Next.js)**
```bash
# تثبيت المتطلبات
npm install

# تشغيل الخادم التطويري
npm run dev

# الوصول للنظام
http://localhost:3000
```

### **2. مولد البطاقات المتقدم**
```bash
# الانتقال للمجلد
cd extension-card-generator

# تشغيل خادم PHP
php -S localhost:8000

# الوصول للنظام
http://localhost:8000          # الواجهة الرئيسية
http://localhost:8000/admin.html  # لوحة التحكم
```

## 🛠️ **التقنيات المستخدمة**

### **Frontend**
- **Next.js 13+** - إطار عمل React متقدم
- **React 18** - مكتبة واجهة المستخدم
- **Tailwind CSS** - إطار عمل CSS
- **Canvas API** - للمحرر المرئي
- **HTML5 & CSS3** - تقنيات الويب الحديثة

### **Backend**
- **PHP 7.3+** - لغة البرمجة الخلفية
- **SQLite** - قاعدة البيانات المحلية
- **GD Library** - معالجة الصور
- **JSON** - تبادل البيانات

### **الميزات المتقدمة**
- **RTL Support** - دعم كامل للعربية
- **Responsive Design** - تصميم متجاوب
- **Progressive Enhancement** - تحسين تدريجي
- **Local Storage** - تخزين محلي للإعدادات
- **AJAX** - تفاعل سلس بدون إعادة تحميل

## 🎯 **الاستخدام**

### **للمستخدمين العاديين:**
1. **افتح الواجهة الرئيسية**: `http://localhost:8000`
2. **املأ بيانات الموظف** (الاسم، المنصب، الصورة)
3. **اختر القالب المناسب**
4. **أنشئ البطاقة وحمّلها**

### **للمديرين:**
1. **افتح لوحة التحكم**: `http://localhost:8000/admin.html`
2. **أدر الحقول والقوالب** حسب الحاجة
3. **خصص التصميم** باستخدام المحرر المرئي
4. **راقب الإحصائيات** والاستخدام
